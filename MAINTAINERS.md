# Maintainers

## Primary maintainer

- Sebastian Legarraga ([@slegarraga](https://github.com/slegarraga))

## Maintainer responsibilities

- Review issues and pull requests.
- Maintain releases and npm package publishing.
- Keep streaming fixtures current for OpenAI, Anthropic, Gemini and compatible endpoints.
- Triage compatibility, security and documentation reports.

## Scope

`llm-sse` is maintained as a public MIT package for provider-portable streaming
event parsing. The primary maintainer controls the GitHub repository, npm
package, release tags and security policy.
